Moderation:
Cinnamon is capable of automatically moderating and deleting comments with a predetermined set of naughty words, even though she
herself is a bit... risque...
- Any user capable of deleting comments can access the "!>chatmod" command.
- "!>chatmod = 1" activates automod, while "!>chatmod = 0" deactivates iit
- "!>chatmod" by itself returns a boolean value: whether or not automod is activated
D&D Dice:
Cinnamon happens to have built in support for D&D rolling with modifiers and advantage/disadvantage with the "roll" keyword!
Basic rolls:
- "roll d6" (not case sensitive) will generate a random number between 1 and 6, inclusive
- "roll d20" will do the same, but with an output between 1 and 20, again, inclusive
Applying modifiers:
- "roll d20 +2" will generate a random number between 1 and 20 (inclusive) and add the value of 2 to the output
- "roll d12 -3" will generate a random number between 1 and 12 (inclusive) and subtract the value of 3 from the output
Advantage and Disadvantage:
- "roll d20 adv" will generate 2 random numbers between 1 and 20 (inclusive) and output the higher value
- "roll d6 dis" will generate 2 random numbers between 1 and 6 (inclusive) and output the lower value
- "roll d20 adv +2" will generate 2 random numbers between 1 and 20 (inclusive), add the value 2 to each, and output the higher number
Detailed Command Descriptions:
Useless/dated commands
- "!>test": returns "Hey, you're not a doofus! Good job, you!"
- "!>SSCSTest": returns "Yep, not a doofus in the slightest ^_^"
- "!>GoodBot": returns "Arigatogozaimasu, "+(sender display name)+"-sama! >.<"
Admin only commands
- "!>chatmod command"
- "!>settings": returns the .config file corresponding to the channel in which the command was enterred
- "!>getLog": Direct messages the sender a copy of the html5 log corresponding to the channel in which the command was enterred
- "!>mute (display name)": mutes a user until either A: The bot crashes, or B: they are manually unmuted
- "!>unmute (display name)": unmutes a user
- "!>get_muted": returns a very unformatted list of currently mutes users
Polls
- "!>get_poll": returns a slightly modified version of the poll file found at "polls/Server.Channel.CinnPoll"
- "!>new_poll (poll name)": clears the poll file found at "polls/Server.Channel.CinnPoll" and replaces it with a blank poll of the desired name
- "!>vote (item)": Increases the value associated with the (item) by 1; if there is no such item in the channel's .CinnPoll, adds the item with a value of 1
Other commands
- "!>superHelp": Attatches this help file to a message in the channel in which the command was enterred
