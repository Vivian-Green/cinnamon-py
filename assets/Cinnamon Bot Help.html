commands

list of commands in cinnamon - these will use the default prefix !> - which may be different in your server

general commands

!>help returns a link to the help page you're looking at

!>test debug command to check if cinnamon can access a channel

!>goodbot debug command to check if cinnamon can read message contents - also to congradulate cinnamon for being cool

!>runtime debug command to check how long cinnamon has been running. Also accessible through the prompt cinnamon, runtime

!>solve see: "cinnamon, eval"

admin commands

!>getlog sends a DM to you with the chat log for this channel

prompts

list of prompts in cinnamon - most of these will require you ask cinnamon directly

dice

the roll command is convenient for the occasional situation where you need to fairly roll some dice

roll

the basic roll prompt - will roll 1 d20

roll d<number of sides>

will roll a die with however many sides you requested. (D10 is handled correctly, 0-9)
try: roll d4

roll <number of dice>d<number of sides>

will roll however many dice with however many sides you requested. (D10 is handled correctly, 0-9)
try: roll 2d8

roll XdXX <"adv" OR "dis">

you can add advantage (adv) or disadvantage (dis) to ANY roll by adding those key words after the roll prompt
try: roll d20 adv

silencing

these commands will make cinnamon ignore the next chat messages

cinnamon, be silenced

cinnamon ignores the next 5 chat messages

good night, cinnamon
or
good night cinnamon

cinnamon ignores the next 50 chat messages

cinnamon, kys or
cinnamon kys or
kys cinnamon or
kys, cinnamon

cinnamon ignores the next 5000 chat messages

evaluates the result of a python expression, and returns it. Useful for quick math.
